HTML/CSS Portfolio Site

Status: Active
Tools Used: HTML, CSS, JavaScript, Nginx
This project is my personal site. The one you are currently visiting. The purpose of this website was simply to create a "digital business card" and showcase some of the projects I have worked on. My goal for this project was to create the site using pure HTML/CSS and JavaScript without using any "pre-built" tools or web builders and also have it look decent on mobile devices. The site is hosted on my linux virtual private server using Nginx.
